# **App Name**: Aurion Resort Digital Experience

## Core Features:

- Hero Section Customization: CMS-driven customization of hero section images/videos and overlay text, without code deployment.
- Intelligent Content Suggestions for Blog Posts: AI tool analyzes existing blog content and suggests relevant topics, keywords, and internal links to improve SEO and user engagement.
- Real-time Availability Display: Display real-time room availability and pricing via API integration with the booking engine.
- Interactive Map Embedding: Embed an interactive Google Map on the contact page for precise location details.
- Newsletter Subscription Management: Collect and manage newsletter subscriptions for targeted marketing campaigns.
- Dynamic Testimonial Display: Fetch guest testimonials dynamically, displaying them in an attractive, rotating carousel.

## Style Guidelines:

- Primary color: Dusty Rose (#D6A7A1), subtly feminine, pairs nicely with resort photography.
- Background color: Ivory (#F5F5F0), creates a soft, clean backdrop.
- Accent color: Deep Navy (#1C2B40), grounding and a good contrast to rose and ivory.
- Body font: 'Literata', serif font, use for comfortable readability and a slightly vintage literary vibe.
- Headline font: 'Playfair', modern serif font with thin-thick lines, imparting an elegant, high-end feel. For longer text, use 'PT Sans' for body.
- Use minimalist, custom icons in gold (#CBA135) to highlight amenities and guide user interactions.
- Emphasize high-resolution photography and videography with full-screen hero sections and gallery layouts.
